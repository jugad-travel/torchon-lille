## Format recommandé

- **Format** : JPG ou PNG
- **Résolution** : Minimum 1200px de largeur pour les images principales
- **Poids** : Optimiser les images (max 500KB par image)
- **Aspect ratio** : 4:3 ou 16:9 pour les images de produits

## Optimisation

Avant d'ajouter les images, pensez à les optimiser pour le web :
- Utilisez des outils comme TinyPNG, ImageOptim ou Squoosh
- Réduisez la taille des fichiers tout en gardant une bonne qualité
- Utilisez le format WebP si possible (avec fallback JPG)
